Quick Start

# Install the SDK
pip install gexa-py

# Use in your code
from gexa_py import Gexa

gexa = Gexa("your-api-key", base_url="http://localhost:8000")

# Search the web
results = gexa.search("AI research papers")

# Get content from URLs
contents = gexa.get_contents(["https://example.com"])

# Get AI-powered answer
answer = gexa.answer("What is machine learning?")

API Endpoints

Endpoint Method Description
